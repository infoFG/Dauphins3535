# 🎨 Charte de couleurs — Les Dauphins sur-le-parc

| Rôle | Nom | Hex | Usage |
|------|-----|-----|-------|
| 🔵 Foncé | Bleu foncé | `#37474F` | Fond header/footer, textes principaux |
| 🟠 Accent | Orange | `#FF9900` | CTA, boutons, éléments interactifs |
| 🩵 Pâle | Bleu pâle | `#D7EFFF` | Arrière-plan sections, cartes |
| 🟤 Burn | Burn | `#3A2404` | Texte sur fonds clairs, variations |
| 🔵 Moyen | Bleu moyen | `#77AAED` | Liens, bordures, icônes |

## Palette CSS

```css
:root {
  --bleu-fonce: #37474F;
  --orange: #FF9900;
  --bleu-pale: #D7EFFF;
  --burn: #3A2404;
  --bleu-moyen: #77AAED;
}
```

## Règles générales

- **Fond sombre** → texte en blanc ou bleu pâle
- **Fond clair** → texte en burn ou bleu foncé
- **Orange** → usage parcimonieux (points d'attention, CTA)
- **Bleu pâle** → toile de fond principale
