# 🎨 Charte de couleurs — Les Dauphins sur-le-parc

| Rôle | Nom | Hex | Usage |
|------|-----|-----|-------|
| ⬛ Fond | Gris foncé | `#606059` | Barre latérale, éléments d'arrière-plan foncés |
| 🟤 Accent | Gold | `#A19B80` | CTA, boutons, éléments interactifs |
| 🩶 Pâle | Gris pâle | `#D3D8D6` | Arrière-plan sections alternées |
| 🟤 Burn | Burn | `#3A2404` | Texte sur fonds clairs |
| 🩹 Moyen | Beige | `#B8B09A` | Liens, bordures, sous-titres |

## Palette CSS

```css
:root {
  --bleu-fonce: #606059;
  --orange: #A19B80;
  --bleu-pale: #D3D8D6;
  --burn: #3A2404;
  --bleu-moyen: #B8B09A;
}
```

## Règles générales

- **Fond sombre** (#606059) → texte en blanc ou beige clair
- **Fond pâle** (#D3D8D6) → texte en burn (#3A2404)
- **Gold** (#A19B80) → usage parcimonieux (CTA, points d'attention)
- **Gris pâle** (#D3D8D6) → toile de fond des sections alternées
